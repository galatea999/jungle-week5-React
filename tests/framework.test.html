Mini React 테스트 보드

이 페이지는 브라우저에서 바로 돌릴 수 있는 기본 스모크 테스트를 실행하고, 아직 assertion으로 옮기지 않은 검증 계획은 시나리오 목록으로 함께 보여줍니다.

최근 추가된 상호작용 smoke test는 실제 button click과 input 이벤트도 확인합니다. 따라서 failed 결과가 보이면, 학습 콘텐츠 문제가 아니라 프레임워크의 이벤트 props 연결 빈칸일 가능성이 큽니다.

아래 시나리오 목록은 이제 `covered`, `blocked`, `pending` 상태를 나눠 보여줍니다. `covered`는 smoke test가 이미 연결된 항목이고, `blocked`는 테스트는 있지만 현재 프레임워크 빈칸 때문에 막힌 항목입니다.
